About Document Translator

This tool translates Chinese text inside .xlsx workbooks, .docx documents, and .pdf files while preserving the original layout as faithfully as possible.

Excel and Word: The file is unzipped, text strings inside the XML parts are translated via the chosen API, and the file is rezipped with translations swapped in. Images, formulas (Excel), tables, formatting, headers, footers, comments — everything else passes through untouched. Word reflows English text in paragraphs automatically, so length differences are handled gracefully.

PDF: The PDF is parsed using PDF.js to extract Chinese text fragments along with their exact positions and font sizes. Translations are then drawn using pdf-lib at the original coordinates, with auto-shrinking and optional wrapping when English text is too long for its original space. All images, vector graphics, and table structure remain in place.

Privacy: Your file never leaves your browser. Only the extracted text strings are sent to your chosen translation provider, using your own API key.

Performance notes: PDFs over ~50 MB or ~200 pages may be slow to process and require significant browser memory. For very large PDFs the chunk size in Step 4 controls memory usage — smaller chunks are slower but safer.

Limitations: Text rendered inside images (e.g. callouts on a JPEG) is not translated — that requires OCR. PDF subset fonts are replaced with Helvetica for English text, so the typeface may not match the original exactly. Vertical text and complex right-to-left text are not specifically handled.
